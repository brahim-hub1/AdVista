# AdVista

AdVista is a SwiftUI iOS app for browsing ad creatives.

**Structure**
```
AdVista/
  App/              App entry point and environment
  Core/             Models, services, and utilities
  DesignSystem/     Shared UI components and styles
  Features/         Feature modules (Overview, CreativesList, CreativeDetail)
  Navigation/       Navigation flows
  Resources/        App resources (including Firebase config)
  Preview/          SwiftUI previews
  Assets.xcassets/  App assets
  Preview Content/  Preview assets
AdVista.xcodeproj/  Xcode project
AdVistaTests/       Unit tests
AdVistaUITests/     UI tests
```

**Build & Run**
1. Open `AdVista.xcodeproj` in Xcode.
2. Select the `AdVista` scheme.
3. Choose an iOS 18.2+ simulator or device.
4. Run with the play button (or `Cmd+R`).
